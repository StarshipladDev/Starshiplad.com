Binary Calculator
The following program is my output for the final "10 Days Of Javascript" challenge on Hackerrank.com .
Requirements
The project had the following requirements to pass: --- Objective In this challenge, we implement a calculator that uses binary numbers. Check out the attached tutorial for learning materials.
Task
Implement a simple calculator that performs the following operations on binary numbers: addition, subtraction, multiplication, and division. Note that division operation must be integer division only; for example, 101(5)/10(2) = 10(2),1111(15) / 100(4) = 11(3).
Only one operation needs to be performed per calculation ,outputs need to be trimmed of any leading zeros.
